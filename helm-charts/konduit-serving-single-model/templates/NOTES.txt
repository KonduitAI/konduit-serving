Get the application URL by running these commands:
export NODE_PORT=$(kubectl get --namespace {{ .Release.Namespace }} -o jsonpath="{.spec.ports[0].nodePort}" services {{ include "pipelines.fullname" . }})
export NODE_IP=$(kubectl get nodes --namespace {{ .Release.Namespace }} -o jsonpath="{.items[0].status.addresses[0].address}")
echo http://$NODE_IP:$NODE_PORT

You may also need to port forward the pod in order to get the server
to show up outside the cluster.
You can do this with:

export POD_NAME=POD=$(kubectl get pod -l app= {{ include "pipelines.fullname" . }} -o jsonpath="{.items[0].metadata.labels[0]}")
kubectl port-forward $POD_NAME {{ .Values.service.port }}:{{ .Values.service.port }}
